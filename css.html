CSS — каскадные таблицы стилей, язык, предназначенный для описания и форматирования внешнего вида веб-страниц. Он позволяет определять стили элементов HTML, таких как шрифты, цвета, отступы и многое другое. CSS (каскадные таблицы стилей) нужен для упрощения процесса придания веб-страницам презентабельного вида. Он позволяет применять стили к HTML-документам, описывая, как должна выглядеть веб-страница, задавая цвета, шрифты, интервалы и расположение. CSS используют для того, чтобы отделить содержание веб-страницы от её оформления. За счёт этого страницы удобнее изменять и поддерживать.  Некоторые задачи, которые решает CSS:

  • Оформление текста. Задание цвета, размера шрифта, выравнивания, высоты строки и т. д..
  • Оформление HTML-элементов. Задание размеров, отступов, цвета фона, теней, скруглений. 
  • Построение сеток для расположения контента. 
  • Создание анимации. С CSS можно создавать красивые анимации без использования JavaScript.
    •  
  • Создание адаптивного дизайна. Можно менять отображение в зависимости от размеров устройства, на котором просматривают сайт. 

История

В начале 1990-х годов, когда интернет только начинал развиваться, веб-страницы были простыми и состояли в основном из текста. Для оформления страниц использовались теги HTML, такие как font, color и center, которые позволяли задавать шрифт, цвет и выравнивание текста. Однако эти теги были ограничены и не позволяли создавать сложные макеты страниц.

В 1994 году компания Netscape представила концепцию каскадных таблиц стилей. Они разработали первый рабочий прототип CSS, чтобы решить проблему с оформлением веб-страниц и сделать их более гибкими и удобными для разработчиков. В 1996 году CSS был официально стандартизирован консорциумом W3C (World Wide Web Consortium).

Первоначально CSS использовался для определения стилей элементов HTML. Он позволял задавать шрифты, цвета, размеры и другие параметры для элементов страницы. Это позволило разработчикам создавать более сложные и красивые веб-сайты.

Со временем CSS стал более мощным и функциональным. Были добавлены новые свойства и значения, а также новые селекторы, которые позволили более точно выбирать элементы для стилизации

CSS-селекторы — это шаблоны форматирования стилей для элементов веб-страниц. Они помогают выбирать определённые элементы страницы, чтобы применить к ним стили или использовать их в коде JavaScript. С помощью селекторов можно задать стили для отдельных элементов и групп элементов, за счёт чего определяется внешний вид страницы.

Также появились новые возможности, такие как градиенты, тени и анимации.

Одним из важных этапов в развитии CSS стало появление CSS3. CSS3 добавил множество новых функций, таких как медиазапросы, гибкие макеты и анимационные эффекты. Это позволило создавать ещё более сложные и интерактивные веб-сайты.

Сегодня CSS является одним из основных языков для веб-разработки. Он используется для создания красивых и функциональных веб-сайтов, приложений и других онлайн-проектов. CSS продолжает развиваться и улучшаться, добавляя новые функции и возможности для разработчиков. Основные современные тенденции в CSS включают:

  • использование препроцессоров, таких как Sass и Less, для упрощения работы с CSS;
  • применение CSS-фреймворков, таких как Bootstrap и Tailwind CSS, для ускорения разработки;
  • использование CSS-анимаций и переходов для создания интерактивных эффектов;
  • внедрение CSS-преобразований для изменения положения, размера и ориентации элементов.

Структура кода

Селектор {
Свойство: значение;
}
Структура CSS включает комментарии, селекторы, свойства и значения. В коде они указываются в таком же порядке. 
Селекторы — это метки, которые помогают браузеру понять, к какой части HTML-кода нужно применить заданные параметры. 
Свойства CSS определяют атрибуты стиля, которые нужно применить к элементам HTML. Они управляют различными аспектами внешнего вида
Способы применения css кода к html-страницам:
1)In-line.
Этот способ предназначен для оформления одного элемента на одной веб-странице. Для применения этого способа нужно написать атрибут “style” после тега элемента, а в кавычках указать необходимые для этого элемента свойства.
Пример кода:
p style=”color:#ff0000;”
2)Внутренний.
Данный способ подходит для оформления нескольких элементов на одной веб-странице.Для применения этого способа после шапки документа нужно написать тег style, внутри которого можно прописывать все оформление данного документа.
Пример кода:
head
style
p {
color:#000000;
}
/style
/head
3)Внешний
Самый эффективный способ при создании многостраничного сайта. Предназначен для оформления нескольких элементов на нескольких веб-страницах.
Подключается при помощи внешнего файла с расширением .css. Создается два (или более) документа: один — для html, другой — для css. В html-файле пишется специальная строчка для связки html-файла с нужным css-файлом. В атрибуте href в кавычках указывается имя css-файла.
Пример кода:
В html-файле:
head
link rel=’stylesheet’ type=”text/css” href=”style.css”/
/head
В css-файле:
p {
color: #ffffff;
}

4)Типы селекторов:

  • Теги

  • Классы

  • Идентификаторы

1.Классы.
Классы предназначены для оформления групп сходных элементов. К 1 тегу может применяться несколько классов. Для создания в html-коде используется атрибут class. В html-файле:
p class=”one” первый класс /p
p class=”two” второй класс /p
В css-файле:
p.one {color: #ff0000; }
p.two {font-size: 20pt; }
2.Идентификаторы.
Идентификаторы предназначены для оформления уникальных элементов. К 1 тегу может применяться только 1 идентификатор. Для создания в html-коде используют атрибут id.
В html-файле:
p id=”one” /p
В css-файле:
#one {color: #ff0000; }
5)Псевдоклассы. Ссылки.
Псевдоклассы определяют динамическое состояние элементов, которое изменяется с помощью действий пользователя, а также положение в дереве документа.
Селектор:Псевдокласс { Описание правил стиля }
1.:active
Происходит при активации пользователем элемента.
Пример:
a: active {
text-decoration: line-through;
}
2.:link
Применяется к непосещенным ссылкам.
Пример:
a:link {
text-transform: uppercase;
}
3.:hover
Активизируется при наведении курсора мыши на элемент, но щелчка по нему не происходит.
Пример:
a:hover {
text-decoration : underline;
}
4.:visited
Применяется к посещенным ссылкам. Обычно такая ссылка меняет свой цвет по умолчанию на фиолетовый, но с помощью стилей цвет и другие параметры можно задать самостоятельно.
Пример:
a:visited {
color: #ffff00;
}
5.:first-child
Применяется к первому дочернему элементу селектора, который расположен в дереве элементов документа.
Пример:
p:first-child {
color: red;
}
Псевдоэлемент.
Псевдоэлементы позволяют задать стиль элементов не определённых в дереве элементов документа, а также генерировать содержимое, которого нет в исходном коде текста.
6.:first-letter
Определяет стиль первого символа в тексте элемента. Это позволяет создавать в тексте буквицу и выступающий инициал.
Буквица представляет собой увеличенную первую букву, базовая линия которой ниже на одну или несколько строк базовой линии основного текста. Выступающий инициал — увеличенная прописная буква, базовая линия которой совпадает с базовой линией основного текста.
6)Позиционирование элементов.
Позиционирование (расположение) элементов задается свойствами: position (тип позиционирования), а также свойствами top (bottom) и left (right), задающими отступы.

Виды позиционирования:
1. position: absolute;
Абсолютное позиционирование. Точно располагает элемент внутри окна браузера или родительского элемента с заданным позиционированием (fixed, relative, absolute).
Пример:
#banner {
position: absolute;
top: 15px;
left: 15px;
}
2. position: relative;
Относительное позиционирование. Изменяет расположение элемента относительного его прежнего положения.
Пример:
img:hover {
position: relative;
top: 15px;
left: 15px;
}

3. position: fixed;
Фиксированное позиционирование. Привязывает элемент к точке в окне браузера и не меняет его положения при прокрутке страницы.
Пример:
#logo {
position: fixed;
top: 15px;
left: 15px;
}

4. position: static;
Значение по умолчанию.
Элементы не имеющие позиционирования (или родительских элементов с позиционированием) располагаются на странице игнорируя положение позиционированных элементов.
7) Блочные и строчные элементы. Создание блоков.
Типы элементов:

  • Строчные – являются частью строки, не обособляются отступами, переносятся на другую строку при необходимости (примеры, img, span, a, b, font).

  • Блочные – размечается в виде прямоугольника, занимает всю доступную ширину, высота определяется содержимым, всегда начинается с новой строки (div, form,
